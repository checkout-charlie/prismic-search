# Welcome Prismic Search 👋

## Install

```sh
yarn install
```

## Usage for development

```sh
yarn dev
```

## Usage for produciton

```sh
yarn build
yarn start
```

## Run tests

```sh
yarn cypress run
```

## Run lint

```sh
yarn lint
```
